You may need to start your config file with one or more the following lines:

  • a line starting with Name if indicated during the EZProxy install process
  • lines beginning with Option and LoginPort if you choose to proxy by hostname 
# Many sites use dx.doi.org when linking from one database to another.
# Useful Utilities recommends leaving this definition in your configuration
# file.  The special -hide option prevents this from showing up in your
# database menu
Title -hide Digital Object Identifier System
URL http://dx.doi.org
Domain doi.org

# Chadwyck Twentieth Century American/English Poetry
Title Chadwyck 20th Century American/English Poetry
URL http://collections.chadwyck.com
Domain chadwyck.com

# EBSCOhost Databases
# http://www.usefulutilities.com/support/db/ebscohost.html
Title EBSCOhost Databases
URL http://search.ebscohost.com/
DJ epnet.com
DJ ebscohost.com
Host search.epnet.com
Find proxy.cfm?url=http://
Replace proxy.cfm?url=http://^A

# eLibrary Curriculum Edition
Title eLibrary Curriculum Edition
URL http://elibrary.bigchalk.com/ce
Domain bigchalk.com

# HeritageQuest
Title HeritageQuest
URL http://www.heritagequestonline.com/
Domain heritagequestonline.com

# History Study Center (Chadwyck)
Title History Study Center
URL http://www.historystudycenter.com/hsc
Domain chadwyck.com
Domain historyonline.com
Domain historystudycenter.com

# Library of Texas
Title Library of Texas
URL http://www.libraryoftexas.org/
Domain libraryoftexas.org

# NetLibrary
Title NetLibrary
URL http://www.netlibrary.com
Domain netlibrary.com

# ProQuest Learning: Literature
Title ProQuest Learning: Literature
URL http://literature.proquestlearning.com/
DJ proquestlearning.com

# Stat!Ref
Title Stat!Ref
U http://online.statref.com/default.aspx?grpalias=tex
DF statref.com

# Texas Digital Sanborn Maps
Title Texas Digital Sanborn Maps
URL http://sanborn.umi.com/
Domain sanborn.umi.com

# Thomson Gale Infotrac Databases
# http://www.usefulutilities.com/support/db/gale.html
Title Thomson Gale Infotrac Databases
URL http://infotrac.galegroup.com/itweb/yyyyyy
Host find.galegroup.com
DJ galegroup.com

# WorldCat
Title WorldCat
URL http://firstsearch.oclc.org/FSIP?dbname=WorldCat
Domain oclc.org

Notes:

  • xxx.xxx.xxx in first line is your ezproxy server domain name (i.e. ezproxy.yourlib.org)
  • yyyyyy in the Thomson Gale config stanza should be replaced with your organization's TexShare member code from Gale
